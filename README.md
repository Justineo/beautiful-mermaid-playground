# Beautiful Mermaid Playground

A minimal live playground for Mermaid diagrams, built with Vue 3.6 + Vite 8.

## Run

```bash
pnpm install
pnpm dev
```

## Scripts

```bash
pnpm lint
pnpm type-check
pnpm build
pnpm preview
```
